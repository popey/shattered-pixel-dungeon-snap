## Shattered Pixel Dungeon

Shattered Pixel Dungeon is a Roguelike RPG, with pixel art graphics and lots of variety and replayability. Every game is unique, with four different playable characters, randomized levels and enemies, and over 150 items to collect and use.

This snap is a community-maintained unofficial build of [Shattered Pixel Dungeon](https://github.com/00-Evan/shattered-pixel-dungeon).

[![Get it from the Snap Store](https://snapcraft.io/static/images/badges/en/snap-store-black.svg)](https://snapcraft.io/shattered-pixel-dungeon)

## Install

Install on any [supported Linux](https://snapcraft.io/docs/installing-snapd) distribution:

```
snap install shattered-pixel-dungeon
```
